## 🚀 How to Deploy

### Option 1: Vercel (Recommended - Fastest)
```bash
cd herco_project
npm install
npm run build
npm install -g vercel
vercel
```

### Option 2: Netlify
```bash
npm run build
npm install -g netlify-cli
netlify deploy --prod --dir=dist
```

### Option 3: Firebase
```bash
firebase init hosting
npm run build
firebase deploy
```

### Option 4: Your Own Server
```bash
npm run build
# Upload dist/ folder to your server
```
